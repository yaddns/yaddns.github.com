Yaddns

Yaddns is a small ddns client with multiple ddns server supports and a high flexibility.

Current stable source is yaddns 0.1.2 yaddns-0.1.2.tar.gz (view changelog). yaddns 0.1.2 support only dyndns.org service and direct wan ip address retrieve.

Next release is yaddns 1.1. It will support changeip, dyndns, dyndnsit, no-ip, ovh and sitelutions services and will have indirect wan ip address retrieve mode. There is an 1.1-rc2 source browsing, yaddns-1.1-rc2.tar.gz (view changelog).

If you have suggestions, want to help, implement new service, ... contact me by email or with github :)
