Yaddns

Yaddns is a small ddns client with multiple ddns server supports and a high flexibility.

Current stable source is yaddns version 0.1.2 yaddns-0.1.2.tar.gz (view changelog).

yaddns 0.1.2 support only dyndns.org service and direct wan ip address retrieve. The next release in developpement, yaddns 1.1, support changeip, dyndns, dyndnsit, no-ip, ovh and sitelutions services and have indirect wan ip address retrieve mode.

Next release is 1.1. There is an 1.1-rc1 source browsing, yaddns-1.1-rc1.tar.gz (view changelog).

If you have suggestions, want to help, implement new service, ... contact me :)
